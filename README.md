# College-Placement-Portal
A web-based Placement Management System developed to streamline the campus recruitment process for students, recruiters, and administrators. The portal provides a centralized platform where students can apply for jobs, companies can post job opportunities, and administrators can manage the entire placement process efficiently.

* Project Overview :-

The College Placement Portal simplifies the placement activities of an institute by digitizing the recruitment process. It reduces manual work and provides an organized system for managing students, companies, and job applications.

* Features :-
-> Student Module
Student Registration and Login
Profile Management
View Available Job Opportunities
Apply for Jobs
Upload Resume
Track Applied Jobs

->Recruiter/Company Module
Company Registration and Login
Create and Manage Job Postings
View Applicants
Download Student Resumes
Manage Recruitment Process

->Admin Module
Manage Students and Companies
Monitor Job Postings
View Portal Statistics
Manage Placement Activities

* Technologies Used :-
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Development Environment: XAMPP Server, Visual Studio Code
